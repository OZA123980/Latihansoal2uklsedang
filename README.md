# Latihansoal2uklsedang
Latihan UKL 2
Program ini menghitung volume tabung dengan menggunakan fungsi yang memiliki parameter jari-jari(r) dan tinggi(t).
Cara kerjannya adalah:
-Pengguna memasukkan nilai jari-jari dan tinggi tabung.
-Nilai tersebut dikirim ke fungsi hitung Volume Tabung(r,t).
-Fungsi menghitung volume berdasarkan rumus di atas dan mengembalikan hasilnya.
-Program utama menampilkan hasil volume tabung ke layar.
